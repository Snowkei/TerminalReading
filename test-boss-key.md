# 老板键功能测试说明

## 功能概述
老板键功能允许用户在阅读时快速隐藏阅读内容，显示一个伪装的工作界面，以便在工作场所或需要隐藏阅读内容时使用。

## 默认快捷键
- 默认老板键：` (反引号)
- 可通过配置文件自定义

## 使用方法
1. 在阅读界面中按下反引号键 (`) 进入老板模式
2. 系统会随机显示以下四种伪装界面之一：
   - 代码编辑器界面
   - 日志文件界面
   - 终端命令界面
   - 服务器状态界面
3. 再次按下反引号键退出老板模式，返回阅读界面

## 测试步骤
1. 启动阅读器：`npm run dev`
2. 选择一本小说开始阅读
3. 按下反引号键 (`) 验证是否进入老板模式
4. 检查显示的伪装界面是否合理
5. 再次按下反引号键验证是否返回阅读界面
6. 在章节列表界面也测试老板键功能

## 自定义配置
可以在配置文件中修改老板键快捷键：
```json
{
  "keyBindings": {
    "bossKey": ["b"]  // 将老板键改为 b 键
  }
}
```

## 注意事项
- 老板键在阅读界面和章节列表界面都有效
- 伪装界面会显示一些随机数据，看起来像真实的工作内容
- 按下老板键后，底部会显示提示信息："按 ` 键返回阅读界面"
